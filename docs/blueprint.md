# **App Name**: MemoryLane Canvas

## Core Features:

- User Authentication: Secure email/password sign-up and login with persistent sessions across devices.
- Scrapbook Creation: Create scrapbooks with titles, categories, and unique IDs.
- Collaborative Editing: Real-time co-editing for up to three users on shared scrapbooks using Firebase Listeners to keep multiple clients in sync with server.
- Media Storage: Upload and store images, short video clips (max. 15 seconds), and audio voice memos organized by user ID.  Storage accounts for user id.
- Canvas Object Management: Enable drag-and-drop system on a creative canvas with touch gestures for resizing and repositioning canvas items. Keep track of objects including their position and scale on the canvas.
- PDF Export: Generate high-quality PDF or JPEG exports of finalized scrapbook canvas layouts for external sharing.

## Style Guidelines:

- Primary color: Soft blue (#A7D1E8), reminiscent of peaceful skies, to convey calm and reflection.
- Background color: Very light desaturated blue (#F4F9FC) to provide a clean, non-distracting backdrop for memories.
- Accent color: Warm peach (#F2B677), a comforting analogous color, used for interactive elements to draw focus.
- Font: 'Alegreya', a serif font, to provide an elegant, intellectual, contemporary feel.
- Use clear, outlined icons to represent different functions, maintaining a modern and user-friendly appearance.
- Employ a flexible grid system for the canvas, ensuring content is responsive and visually balanced.
- Implement subtle animations and transitions for user interactions, such as dragging, dropping, and resizing elements on the canvas.
